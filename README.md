# 🚀 AI-Powered Rocket.Chat App Generator CLI

A simple and beginner-friendly CLI tool to automatically generate a basic Rocket.Chat app template with minimal input.

---

## 📌 Overview

Building apps for Rocket.Chat can be complex for beginners due to its architecture and setup process.

This tool simplifies the workflow by generating a ready-to-use app structure instantly using a command-line interface.

---

## ✨ Features

- 📥 Takes user input (App Name, Description, Author)
- 📁 Automatically creates project folder
- 📄 Generates essential files:
  - app.json
  - index.ts
  - README.md
- ⚠️ Handles errors (empty input, existing folder)
- ✅ Beginner-friendly implementation using basic Python

---

## 🛠️ Technologies Used

- Python (Basic)
- File Handling
- CLI (Command Line Interface)
- Git & GitHub

---

## ⚙️ How It Works

1. Run the Python file  
2. Enter required details  
3. Tool generates a complete app template  
4. Ready to use or extend further
